Project information

The E-Commerce Management System is a database-based web application that allows users to register, log in, browse products, search for items, add them to a cart, and place orders.
Admins can add, update, and delete products.
The system uses MySQL triggers, procedures, functions, and relational schema concepts to manage data efficiently.

Key Features

User Registration & Login
View and Search Products
Add to Cart & Update Cart
Place Orders
Admin Product Management
DB-Level Triggers for Admin Restrictions
Audit Logging for Product Updates/Deletes
